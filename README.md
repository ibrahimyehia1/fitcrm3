# FitCRM

**Author:** Ibrahim Yehia

## Description
FitCRM is a front-end project that simulates a simple CRM system for fitness clients.

It includes two pages:
1. **New Client Form (index.html)** — A form with labeled fields and placeholders for client details.
2. **Client List (clients.html)** — A responsive table displaying 10 sample clients with a search box and placeholder Edit/Delete buttons.

The form and buttons are placeholders only (no backend functionality).

## Tech Stack
- HTML5
- CSS3 (Flexbox and responsive layout)

## Deployment
Hosted using GitHub Pages.

If deployment doesn’t work, open the project locally:
1. Download the repository
2. Open `index.html` in your browser
